# roadstop-scraper

全国の道の駅、および高速道路のSA(サービスエリア)・PA(パーキングエリア)に関する位置情報・名称・付加情報を対象Webサイトからスクレイピングして収集し、GeoJSON形式で保存するデータ収集アプリケーション。

## 概要

- 道の駅・SA/PAの位置情報(緯度経度)、名称、付加情報のスクレイピング
- 収集結果のGeoJSON形式での永続化(都道府県単位でファイル分割)
- 中断・再開可能なレジューム機能
- サードパーティサーバへの負荷を抑えるリクエスト頻度制御(レート制限)
- HTTP取得・HTMLパース・リトライ/レート制限・構造変化検知を提供するスクレイピングエンジン(`scraping`パッケージ、道の駅/SA・PA個別スクレイパの共通基盤)
- 道の駅スクレイパー(`michinoeki`パッケージ)は全国・地方(8区分)・都道府県単位での対象範囲指定に対応し、対象サイトの一覧から消失した道の駅を即座に削除せず削除状態(削除フラグ・最終確認日時)を伴って保持したうえで、1年経過後に完全除去する削除状態管理を持つ

## セットアップ

```bash
pdm install
```

## 開発コマンド

```bash
# Lint
pdm run ruff check .

# テスト(HTMLカバレッジレポートを report/ に出力)
pdm run pytest
```

## 実行コマンド

```bash
# 全国の道の駅を対象に収集し、都道府県単位のGeoJSONへ出力
michinoeki-scrape

# 地方区分を指定して対象を絞り込む(例: 関東)
michinoeki-scrape --region kanto

# 都道府県コードを指定して対象を絞り込む(例: 北海道)
michinoeki-scrape --prefecture-code 01
```

`--region`・`--prefecture-code`は同時指定不可で、いずれも省略した場合は全国が対象となる。指定した値が存在しない場合はエラーメッセージを表示して非ゼロの終了コードを返し、この場合HTTPリクエストは発生しない。

```text
├── geo-json/                # スクレイピング結果(GeoJSON)の出力先

```
